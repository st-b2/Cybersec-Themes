# 🕵️ Руководство по анонимности в сети

## Введение
Анонимность в интернете — это не просто использование одного инструмента, а комплексный подход к защите личности и данных. Это руководство охватывает базовые и продвинутые методы сохранения анонимности.

---

## 🔐 Основные принципы (OPSEC)

1. **Минимизация цифрового следа** — не оставляйте лишней информации о себе.
2. **Разделение личностей** — используйте разные аккаунты, браузеры и даже устройства для разных целей.
3. **Постоянное обновление знаний** — угрозы меняются, и защита должна эволюционировать.
4. **Доверяй, но проверяй** — не верьте обещаниям «полной анонимности» от бесплатных сервисов.

---

## 🌐 Tor (The Onion Router)

### Что такое Tor?
Tor — это сеть, которая маршрутизирует ваш трафик через несколько узлов (ноды), скрывая ваш реальный IP-адрес.

### Установка и настройка
1. Скачайте официальный **Tor Browser** с [torproject.org](https://www.torproject.org/).
2. Установите и запустите. По умолчанию он уже настроен.
3. **Важно:** не устанавливайте дополнительные расширения — это может нарушить анонимность.
4. Используйте **новый цепь** (New Identity) для смены идентификатора.

### Советы по использованию Tor
- Не заходите в личные аккаунты (почта, соцсети) через Tor, если они привязаны к вашей реальной личности.
- Используйте **HTTPS везде**, где возможно.
- Отключите JavaScript для повышения безопасности (в настройках безопасности Tor Browser можно выставить уровень «Самый высокий»).

## 🛡️ VPN (Virtual Private Network)

### Зачем нужен VPN?
VPN шифрует весь ваш интернет-трафик и маскирует IP-адрес, но он **не делает вас полностью анонимным**, особенно если логи хранятся.

### Выбор VPN-провайдера
| Критерий | Что искать |
| :--- | :--- |
| **Политика логирования** | Zero-log (не хранят логи) |
| **Юрисдикция** | Страны без обязательного хранения данных (например, Панама, Швейцария) |
| **Шифрование** | AES-256, OpenVPN / WireGuard |
| **Платёж** | Возможность оплаты криптовалютой (Monero, Bitcoin) |
| **Количество серверов** | Больше серверов = меньше нагрузка |

⚠️ **Не используйте бесплатные VPN** — они часто продают данные или показывают рекламу.

## 🧩 Дополнительные инструменты

### Прокси-серверы (SOCKS5, HTTP)
Могут использоваться в связке с Tor (например, Tor → прокси) или для обхода географических блокировок, но **не дают анонимности** без других мер.

### I2P (Invisible Internet Project)
Сеть, похожая на Tor, но ориентированная на скрытые сервисы внутри сети. Используется для сайтов в домене `.i2p`.

### Freenet
Децентрализованная сеть для публикации контента без цензуры.

## 🌍 Браузеры и настройки

### Firefox (с дополнениями)
- Установите **uBlock Origin** и **Privacy Badger**.
- Включите **Do Not Track**.
- Отключите WebRTC (утечка IP через WebRTC) – можно через about:config, установите `media.peerconnection.enabled = false`.

### Brave
Встроенная защита от трекеров, но есть встроенная реклама (можно отключить).

### Tor Browser (рекомендуется для максимальной анонимности)
Используйте его по умолчанию для всех действий, где требуется анонимность.

---

## 🔑 Управление паролями и двухфакторная аутентификация

- Используйте менеджер паролей (Bitwarden или др.) с генерацией сложных паролей.
- Включайте **2FA** (TOTP) везде, где возможно, но используйте отдельное приложение (Google Authenticator или др.).
- Для особо важных аккаунтов используйте **аппаратные ключи** (YubiKey).

---

## 📱 Мобильная анонимность

- Используйте **Orbot** (Tor для Android) и **Onion Browser** для iOS.
- На телефоне отключайте геолокацию для приложений, где она не нужна.
- Используйте VPN в связке с Tor (VPN → Tor или Tor → VPN — варианты с разными уровнями безопасности).

---

## 🗑️ Очистка данных

- Регулярно очищайте историю, куки и кэш браузера.
- Используйте **CCleaner** или встроенные средства очистки.
- Для полного уничтожения файлов используйте **shred** (Linux) или **Eraser** (Windows).

---

## 🧠 Поведенческая анонимность

- Не используйте один и тот же никнейм на разных платформах.
- Не публикуйте метаданные (Exif) в фото — удаляйте их перед загрузкой.
- Не рассказывайте о своих привычках и времени активности в сети.
- Используйте разные почтовые ящики для разных сервисов.

---

## 🚨 Важные предупреждения

- Анонимность — не абсолютна. Даже при всех мерах правоохранительные органы могут идентифицировать вас при грубых ошибках.
- Не доверяйте сервисам, которые обещают «100% анонимность» — это маркетинг.
- Законодательство разных стран может запрещать использование Tor или VPN. Проверяйте местные законы.

## [ДОПОЛНИТЕЛЬНО] 🌟 Как добиться аннонимности GOD Mode

В качестве Double VPN можно использовать Nord VPN , Mullvad VPN или Proton VPN. За их безопасность можно не сомневаться. 
При этом вам не будет необходимости устанавливать Linux и делать маршрутизацию между 
серверами вручную.

Схема представляет из себя:

Клиент -> Double VPN* -> Удаленное рабочее место (например, можно использовать старый ноутбук) + Виртуальная машина -> VPN -> Цель

- "Double VPN*" - Рядом друг с другом, но в разных дата-центров
 
- Double VPN, или процесс сцепления VPN-серверов, — это технологическое решение, 
которое позволяет скрывать вашу онлайн-активность за несколькими серверами через 
туннель VPN. На самом деле принцип работы довольно прост: вы подключаетесь к 
первому серверу VPN, который в свою очередь перенаправляет весь трафик на другой 
сервер, с которого Интернет-трафик достигает конечного пункта назначения. 
Таким образом, соединение идет через два отдельных сервера, расположенных в разных 
местах: внешний IP-адрес меняется, и трафик шифруется один раз, а затем он шифруется 
еще раз на втором сервере в другом месте.
