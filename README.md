# ⚔ Библиотека ИБ & Хакинга

Добро пожаловать в открытую библиотеку по ИБ!  

Этот репозиторий создан для систематизации знаний в области этичного хакинга, пентеста, анализа защищенности и цифровой анонимности. Здесь вы найдете:

- 📖 Книги и мануалы по Kali Linux, Python, веб-безопасности, API, Active Directory, Wi-Fi атакам и OSINT.
- 🧾 Чит-листы по Nmap, SQL-инъекциям, реверс-шелам, Metasploit и Linux-командам.
- 📄 Методологии пентеста — от разведки до составления отчетов.
- 🔐 Гайды по анонимности и защите личных данных.

Материалы подходят как для начинающих, так и для опытных специалистов. 

⚠ Все знания — исключительно в образовательных целях.
