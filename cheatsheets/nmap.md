# 🗺️ Nmap Cheat Sheet

## Основные флаги

| Флаг | Описание | Пример |
| :--- | :--- | :--- |
| `-sS` | SYN-сканирование (скрытое) | `nmap -sS 192.168.1.1` |
| `-sT` | TCP-сканирование | `nmap -sT 192.168.1.1` |
| `-sU` | UDP-сканирование | `nmap -sU 192.168.1.1` |
| `-sV` | Определение версий служб | `nmap -sV 192.168.1.1` |
| `-O` | Определение ОС | `nmap -O 192.168.1.1` |
| `-p` | Указать порты | `nmap -p 80,443 192.168.1.1` |
| `-p-` | Все порты (1-65535) | `nmap -p- 192.168.1.1` |
| `-A` | Агрессивное сканирование | `nmap -A 192.168.1.1` |
| `-Pn` | Не пинговать | `nmap -Pn 192.168.1.1` |
| `-sC` | Стандартные скрипты | `nmap -sC 192.168.1.1` |

## Популярные сканы

```bash
# Быстрый сканирование самых популярных портов
nmap -T4 -F 192.168.1.1

# Сканирование с определением версий и ОС
nmap -sV -O 192.168.1.1

# Сканирование всех портов с агрессивным режимом
nmap -p- -A 192.168.1.1
```
## Скрипты NSE
```bash
# Сканирование с использованием скриптов
nmap --script=vuln 192.168.1.1

# Сканирование подсети
nmap 192.168.1.0/24

# Уязвимости
nmap --script=vuln 192.168.1.1

# SMB
nmap --script=smb-enum-shares 192.168.1.1

# HTTP
nmap --script=http-enum -p 80 192.168.1.1

# Heartbleed
nmap --script=ssl-heartbleed -p 443 192.168.1.1

# EternalBlue
nmap --script=smb-vuln-ms17-010 -p 445 192.168.1.1
```
## Сохранение
```bash
nmap -oN scan.txt 192.168.1.1
nmap -oX scan.xml 192.168.1.1
nmap -oA scan 192.168.1.1
```
